Documentation

1. Title:

A Movies Web Application

2. What the API does:

It will provide users with access to information about different movies, directors, and genres

3. List of URL endpoints and request methods

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of all movies /movies GET None A JSON object holding data about all the movies
Return data about a single movie /movies/:Title GET None A JSON object holding data about a single movie. Example:
Return data about a genre /genre/:Name GET None A JSON object holding data about a genre
Return data about a director /director/:Name GET None A JSON object holding data about a director
Allow new users to register /users POST A JSON object holding data about the user to add, structured like: 
                    {
                        "Username": "gamoore7",
                        "Password": "bookbinder6",
                        "Email": "gerald.moore42@gmail.com",
                        "Birth": "1989-12-18"    
                    }
A JSON object holding data about the user that was added, structured like: 
                        {
                            "Username": "gamoore7",
                            "Password": "bookbinder6",
                            "Email": "gerald.moore42@gmail.com",
                            "Birth": "1989-12-18T00:00:00.000Z",
                            "FavoriteMovies": [],
                            "_id": "64bc52d58bf19d6302425352",
                            "__v": 0
                        }
Allow users to update their user info /users/:Username PUT A JSON object holding data about the user to update, structured like: 
                        {
                            "Username": "gamoore7",
                            "Password": "updatedpassword",
                            "Email": "gerald.moore42@gmail.com",
                            "Birth": "1989-12-18"    
                        }
A JSON object holding data about the user that was updated, structured like: 
                            {
                                "_id": "64bc52d58bf19d6302425352",
                                "Username": "gamoore7",
                                "Password": "updatedpassword",
                                "Email": "gerald.moore42@gmail.com",
                                "Birth": "1989-12-18T00:00:00.000Z",
                                "FavoriteMovies": [],
                                "__v": 0
                            }
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST None Send user a message that the new movie has been added to their list
Allow users to remove a movie to their list of favorites /users/:Username/movies/:MovieID DELETE None Send user a text message informing them that the movie has been deleted from their list
Allow existing users to de-register /users/:Username DELETE None Send a text message indicating that the user has been deleted
